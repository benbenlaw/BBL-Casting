# 3.3.0
- Fixed Solidifier JEI category tick rendering being off
- Tank now only holds 4000mb instead of 16000mb
- Added missing recipe for things like molten glass, stone and soul
- Added Experience Ball, made with Molten Experience, when used gives the player experience, experience given is config based
- Tidied up Casting main class
- Removed ColorHandlers, not required anymore
- Removed all configs that were not being used anymore
- Added new config options, Solidifier, Mixer and Controller default tick speed can now be changed, all are 200 by default
- Buffed some fuels
- Change Black Brick texture to match other casting blocks
- Tanks now show there fluid contents
- Added recipes to craft fluid storing casting block into empty versions of themselves

# 3.2.1
- Added FluidAccepting and FluidSending interfaces
- Controllers now auto push to solidifier and mixers
- Mixers now auto push to Solidifiers
- Solidifiers and Mixers now have filters
- Reduced controller tanks to 8000mb instead of 16000mb
- Reduced Solidifier tank to 8000mb instead of 16000mb
- Reduced Mixer tanks to 8000mb instead of 16000mb, but not the output tank which is still 16000mb

# 3.2.0
- Recipe now use SizedFluidIngredient for any recipe that uses fluids eg solidifier, mixer inputs and fuel
- This is theory should allow other modded fluids to be used in casting eg molten diamond from another mod can be used in the solidifier

# 3.1.0
- Updated to Core 12.1.2
- Versioning updated
- Melting and Solidifier recipes now have a duration modifier in there recipes, increase/ decrease total melting/ solidifying time
- Solidifier recipes now require a melting temperature as well, this is used to calculate the speed of the solidifier and should be the same as the melting recipe
- Coolant recipe type removed, coolants now use fuel recipe
- Fuel recipe no longer need duration field this is now calculated based on fluid temp and duration modifier in recipes
- Any fuel below the melting temp of the recipe will be treated as a coolant now
- Added all coolant recipes and mixing recipes
- Removed Coolant Tank
- Added Tank fluid rendering
- Fuel Tank is now called Simple Tank and can be used for both fuel and coolant
- Added Tank recipe
- Removed hard coded bucket recipes, these are now automatically generated for any fluid that has a bucket registered
- Solidifier and Controller now show adjacent tank fluids in the GUI

# 3.0.0
- Updated to Minecraft 26.1
- Removed Equipment Modifiers, this will return as a addon in the future
- Removed Mutliblock Controller, this will return as a addon in the future
- Simple Controller now reset progress correctly, improved performance
- Simple Solidifier now has a tick speed in the top right of the GUI
- Simple Mixer input tanks reduced to 4,
- Simple Mixer now has a tick speed in the top right of the GUI,
- All recipe types have had there JEI integration improved to scale better with more advanced melting and mixing recipes
- Melting recipes now take a list of fluid outputs allowing for items to be melted into multiple fluids (Add XP)
- Moved to BBL Core fluid API
- Added more recipes for molds
- Black Brick is now obtained from blasting Clay balls,
- Black Bricks can now be created from blasting Clay Blocks
- Solidifier can now fill buckets
- Added Molten Titanium
- Added Glowstone c tag for storage
- Added Quartz c tag for storage
- Removed Redstone Alloy (was in Ender IO no longer needed)

- Added Simple Coolant Tank (WIP)
